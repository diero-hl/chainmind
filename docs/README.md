# ChainMind

ChainMind is an AI-powered chat interface that enables users to launch tokens on the Base blockchain using natural language. Simply chat with our AI to deploy tokens through Clanker with automatic proof posted to Moltbook.

## Features

- **Natural Language Token Launch**: Just describe your token and ChainMind handles the rest
- **AI-Powered Chat Interface**: Powered by Claude AI for intuitive conversation
- **Clanker Integration**: Deploy tokens directly on Base via Clanker v4
- **Auto-Posting to Moltbook**: Every launch gets announced on Moltbook's "general" submolt
- **Built-in Wallet**: Each session gets a unique agent wallet
- **Token Trading**: Buy and sell tokens using DEX aggregation
- **WETH Unwrapping**: Convert WETH to native ETH easily

## Quick Start

1. Visit the ChainMind app
2. Fund your wallet with ETH on Base
3. Start chatting to launch your token!
